## 4. 데이터 소스 및 접근 전략

### 소스 1: 국립국어원 한국어기초사전 API (krdict)
- **URL**: https://krdict.korean.go.kr/api/search
- **필요**: API 인증키 (이메일 등록으로 발급)
- **활용**: 한국어 표제어 → 다국어 번역 (영어/베트남어/중국어 등 10개 언어 지원)
- **일일 제한**: 50,000건
- **전략**: `data/scripts/krdict_crawler.py` 작성 → API 키 입력 후 실행
- **기대**: 대부분의 기초 과학 용어는 사전에 등재되어 있음

### 소스 2: 중앙다문화교육센터 전자책 (edu4mc)
- **URL**: https://www.edu4mc.or.kr/edu/list.html
- **자료**: "스스로 배우는 교과 속 어휘" 시리즈 (초등 3-4학년 과학)
- **형식**: PDF
- **전략**: `data/scripts/pdf_extractor.py` 작성 → PDF 다운로드 후 텍스트 추출
- **기대**: 교과 맥락에 맞는 쉬운 설명 + 다국어 대역

### 소스 3: 교과서 및 교육과정 문서 (수동 보완)
- 금성출판사 초등 과학 교과서 단원 목차
- 2022 개정 교육과정 과학과 성취기준
- 교사 경험 기반 핵심 용어 선별

## 5. 우선 작업 순서
1. ✅ PLAN.md 작성 (이 파일)
2. `krdict_crawler.py` — API 크롤러 (키 필요, 실행 준비)
3. `pdf_extractor.py` — PDF 추출기 (파일 필요, 실행 준비)
4. `science_3-4.json` — 교육과정 분석 + 검증된 번역으로 직접 구축
5. 자기 검증 → `VALIDATION.md`

## 6. 품질 기준
- 모든 용어에 `easy_ko` (쉬운 한국어 설명) 필수
- `translations`에 vi, zh, en 3개 언어 필수
- 중복 term_ko 없음
- 과학 분야 전문 번역 정확성 우선 (일반 번역 X)
