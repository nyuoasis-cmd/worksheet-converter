## 1. 현황 및 정책 배경

### 1.1 이주배경학생 현황 (교육부 공식 통계)

| 연도 | 이주배경학생 수 | 전체 학생 대비 |
|------|--------------|--------------|
| 2014 | 68,000명 | 1.1% |
| 2023 | 181,000명 | 3.5% |
| 2026 예상 | 계속 증가 추세 | 4% 이상 |

- 지역 집중: 경기(27%), 서울(11%)
- 한국어학급: 2019년 326개 → 2023년 527개 (1.6배 증가), 수용 인원은 수요 대비 10.3%에 불과

### 1.2 법적 근거

- 2023년 10월 「초·중등교육법」 개정 → 국가·지자체의 다문화교육시책 수립 의무화
- 교원 연수: 담당교사 연 15시간, 학생 연 2시간 다문화교육 의무화

### 1.3 이주배경학생 유형 (교육부 공식 분류)

| 유형 | 정의 |
|------|------|
| 국내출생 자녀 | 한국인 + 결혼이민자 사이에서 국내 출생·성장 |
| 중도입국 자녀 | 재혼 후 본국에서 데려온 자녀, 또는 이민자 본국에서 성장하다 입국한 자녀 |
| 외국인가정 자녀 | 외국인 사이에서 태어난 경우 (한국계 중국인, 고려인, 시리아 난민 등 포함) |

## 2. 다문화교육 정책학교 운영 체계 (교육부 공식 가이드라인)

### 2.1 학교 유형

| 유형 | 대상 | 핵심 내용 |
|------|------|----------|
| 다문화유치원 | 전체 유아·학부모·교사 | 언어발달 지원 + 다문화교육 프로그램 연중 운영 |
| 다문화 중점학교 (초·중등) | 전체 학생 | 교과 연계 다문화교육 + 맞춤형 학생 지원 |
| 한국어학급 | 중도입국·외국인학생 다수 재학 시 | 별도 학급 개설 → 한국어 집중교육 |
| 다문화 연구학교 | 교육부 지정 학교 | 새로운 수업모델 개발·보급 |
| 글로벌 선도학교 | 일반 학생 다수 | 세계시민의식·국제이해교육 강화 |

### 2.2 운영 구조 (5개 영역)

```
다문화교육 정책학교
│
├── A. 다문화교육 (전체 학생 대상)
│   ├── 교과 연계 수업
│   ├── 창의적 체험활동 (자율·동아리·봉사·진로)
│   └── 기타 활동 연계 (독서골든벨, 학교 축제 부스, 세계인의 날 주간 등)
│
├── B. 다문화학생 맞춤형 교육
│   ├── 기초학력 향상 (방과후, 멘토링)
│   ├── 진로지도 (커리어넷, 원격영상 멘토링)
│   ├── 정서 지원 (다국어 검사, 개인·그룹 상담)
│   └── 이중언어교육 (동아리, 방과후, 이중언어대회)
│
├── C. 다문화 친화적 환경조성
│   ├── 물리적 환경 (게시판, 교구·도서 비치, 유휴공간 활용)
│   └── 다문화 급식 ('다문화 음식의 날')
│
├── D. 교원 역량 강화
│   ├── 원격연수: 기초 15차시 / 심화 30차시
│   └── 학습공동체 운영
│
└── E. 가정·지역사회 연계
    ├── 다문화가정 학부모 연계 프로그램
    └── 가족센터·대학·유관기관 연계
```

### 2.3 다문화교육 6대 핵심 요소

| 요소 | 내용 |
|------|------|
| **평등성** | 인종·성·계층과 무관하게 모두 평등하다는 태도와 가치 |
| **반편견** | 선입견·편견·고정관념에 대한 비판적 사고와 대처 능력 |
| **정체성** | 긍정적 자아개념 + 문화적 정체성의 유연한 접근 |
| **다양성** | 다양한 개인·집단의 차이를 알고 존중 |
| **문화이해** | 각 문화를 맥락 속에서 이해, 차이를 이해·존중 |
| **협력** | 다양한 사람들과 상호작용·공동 목표 달성 능력 |

#### 유치원 (만 3~5세) — 누리과정 연계

### 5.1 다문화이해교육 수업사례 부문 수상작

## 8. 수업 방법론 정리

현장 우수사례 및 공식 가이드라인을 통해 확인된 효과적 수업 방법:

| 방법 | 설명 | 주요 적용 사례 |
|------|------|-------------|
| **PBL (문제기반학습)** | 실제 다문화 문제를 팀이 해결 | 공존의 교실 (Design Thinking) |
| **프로젝트 수업** | 주제 중심 장기 탐구 | 세계로 떠나는 문화탐험대, 나비효과 프로젝트 |
| **주제 중심 학습** | 단원 경계 없이 다문화 주제로 통합 | 다문화 공간 탐구, 지역 이야기 |
| **창의 융합** | 여러 교과 요소를 다문화 주제로 결합 | 다문화 창의 융합 프로젝트 |
| **문예 창작** | 시 쓰기·편지 쓰기·그림책 만들기 | 들꽃네아이들 프로젝트 |
| **영상 감상 + 토의** | 다문화 관련 영화·영상을 보고 토의 | 샤방샤방 샤랄라 활용 수업 |
| **지역 탐방** | 지역 내 다문화 공간 직접 탐방 | 다문화를 담은 길안내 |
| **K-POP 활용** | 대중음악을 통한 한국어 학습 | K-POP 한국어 의사소통 |
| **읽기 전략 + 그림책** | 그림책으로 언어·문화 동시 학습 | 읽기 전략 기반 그림책 |
| **역할극·체험놀이** | 다른 나라 문화를 몸으로 체험 | 티니클링, 전통 놀이 |
| **독서 골든벨** | 다문화 주제 도서 활용 독서 행사 | 학교 행사 연계 |
| **다문화 부스 운영** | 학교 축제에 나라별 문화 체험 부스 | 세계인의 날 주간 연계 |
