3학년 1학기

자석(nam châm / magnet)에 붙는 물건만 고른 것은 무엇인가요?

① 클립(kẹp giấy / clip), 쇠못(đinh sắt / iron nail)

② 나무젓가락(đũa gỗ / wooden chopsticks), 지우개(cục tẩy / eraser)

③ 클립(kẹp giấy / clip), 유리컵(cốc thủy tinh / glass cup)

④ 지우개(cục tẩy / eraser), 쇠못(đinh sắt / iron nail)

자석(nam châm / magnet)의 같은 극(cực / pole)(= 같은 쪽)끼리 가까이 하면 어떻게 되나요?

① 서로 당겨요.

② 서로 밀어내요.

③ 아무 일도 안 생겨요.

④ 자석(nam châm / magnet)이 부러져요.

나침반(la bàn / compass)(= 방향을 알려주는 도구) 바늘이 가리키는 방향(hướng / direction)은 어디인가요?

① 동쪽(phía đông / east)과 서쪽(phía tây / west)

② 남쪽(phía nam / south)과 북쪽(phía bắc / north)

③ 위쪽(phía trên / up)과 아래쪽(phía dưới / down)

④ 오른쪽(bên phải / right)과 왼쪽(bên trái / left)

다음 중 자석(nam châm / magnet)을 사용하지 않은 것은 무엇인가요?

① 냉장고(tủ lạnh / refrigerator) 문

② 필통(hộp bút / pencil case) 뚜껑

③ 나무(gỗ / wood) 의자

④ 자석(nam châm / magnet) 칠판

자석(nam châm / magnet)의 N극(cực Bắc / North pole)(= 북쪽 극)과 S극(cực Nam / South pole)(= 남쪽 극)에 대한 설명 중 틀린 것은 무엇인가요?

① 자석(nam châm / magnet)에는 N극(cực Bắc / North pole)과 S극(cực Nam / South pole)이 있어요.

② N극(cực Bắc / North pole)과 S극(cực Nam / South pole)은 서로 당겨요.

③ 같은 극(cực / pole)(= 같은 쪽)끼리는 서로 당겨요.

④ 자석(nam châm / magnet)을 반으로 자르면 새로운 N극(cực Bắc / North pole)과 S극(cực Nam / South pole)이 생겨요.
